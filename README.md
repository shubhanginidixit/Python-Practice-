
 # Python Practice

## Overview

This repository contains a collection of Python practice programs covering various programming concepts such as decorators, generators, object-oriented programming, exception handling, data analysis, and data visualization.

## Topics Covered

* Python Classes and Objects
* Decorators and Generators
* Exception Handling
* Logging and Profiling
* File Handling
* Functional Programming (`map`, `filter`, `lambda`)
* Data Analysis using Pandas
* Data Visualization using Matplotlib

## Libraries Used

* Pandas
* Matplotlib
* Logging
* cProfile

## Purpose

The purpose of this repository is to practice and demonstrate different Python programming concepts through small practical examples and scenarios.

## Example Programs

* Mean, Median, Mode using Pandas
* Bar Plot and Scatter Plot using Matplotlib
* Custom Decorators for Logging and Validation
* Exception Handling Programs
* Generator Functions

## Conclusion

This repository serves as a learning resource for improving Python programming skills and understanding practical implementations of core Python concepts.
